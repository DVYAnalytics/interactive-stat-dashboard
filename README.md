# 📊 StatPulse - Interactive Data & Predictive Dashboard

## ✨ Key Features

- **Flexible Data Ingestion:** Upload custom CSV files or explore using the integrated California Housing dataset.
- **Exploratory Data Analysis (EDA):** Instant overview metrics (row count, feature dimensions, missing value checks) and interactive summary statistical tables.
- **Dynamic Data Visualizations:** Custom scatter plots with optional categorical coloring, fitted OLS trendlines, and interactive correlation heatmaps powered by Plotly.
- **Predictive ML Simulator:** Train and configure a **Random Forest Regressor** in real time with configurable parameters (test set ratio, tree counts) and live performance evaluation ($R^2$ score and RMSE).

## 🛠️ Tech Stack & Dependencies

- **Framework:** Streamlit
- **Data Manipulation:** Pandas, NumPy
- **Machine Learning:** Scikit-Learn
- **Data Visualization:** Plotly Express, Plotly Graph Objects
